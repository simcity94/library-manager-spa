# 📚 LibraryManager SPA

A modern Single Page Application (SPA) built with **lit-html** and **page.js**, designed to manage users, books, and activities using a modular architecture.

---

## 🚀 Overview

This project demonstrates a clean SPA structure with client-side routing and reusable components. It serves as a foundation for building full CRUD applications with authentication.

---

## 🧩 Features

* ⚡ Client-side routing with page.js
* 🧱 Component-based rendering with lit-html
* 🗂 Modular architecture (views & layout separation)
* 📄 Multiple views:

  * Home
  * Users
  * Books
  * Activities
  * Login / Register
* 🎯 Clean UI with responsive layout

---

## 🛠 Tech Stack

* JavaScript (ES Modules)
* lit-html
* page.js
* HTML & CSS

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/simcity94/library-manager-spa.git
   ```

2. Open `index.html` in your browser
   OR use Live Server (recommended)
